# Password Strength Checker

A simple and efficient web application that analyzes the strength of a password in real time.

This project evaluates user passwords based on multiple security criteria such as length, uppercase and lowercase characters, numbers, and special symbols. The goal is to help users understand how secure their password is and encourage better password practices.

## Features

* Real-time password analysis
* Strength scoring system
* Visual feedback (weak, medium, strong)
* Clean and responsive user interface
* Lightweight and fast

## How It Works

The application checks the password against common security rules:

* Minimum length requirement
* Use of uppercase and lowercase letters
* Inclusion of numeric characters
* Inclusion of special characters

Based on these checks, the system calculates a strength score and provides instant feedback to the user.

## Purpose of the Project

This project was created to demonstrate practical front-end development skills and basic cybersecurity concepts. It focuses on usability, performance, and security awareness.
