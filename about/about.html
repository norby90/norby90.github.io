About this project

Built to make password security simple for everyone.

PSM started as a school project and grew into a fully functional tool that helps people understand and improve their password habits — in real time, with no confusion.

The project

🔐

What is PSM?

Password Strength Manager is a web application that analyzes your password in real time, shows you exactly what's missing, and estimates how long it would take for a modern computer to crack it.

⚡

How it works

As you type, PSM checks your password against 5 core security rules — length, lowercase, uppercase, digits, and symbols — and calculates your score instantly, no server involved.

🛡️

Your privacy

Your password never leaves your browser during the strength check. When you create an account, your password is hashed with bcrypt before being stored — we never see it in plain text.

🎓

Why we built it

Most people don't know what makes a password strong. PSM was built to close that gap — to give clear, visual, real-time feedback that anyone can understand and act on immediately.

Why it matters

81%

Of breaches involve weak passwords

The majority of data breaches aren't caused by sophisticated hacking — they exploit passwords that are too short, too simple, or reused across multiple accounts.

3s

To crack "password123"

A modern GPU can test billions of password combinations per second. A simple, predictable password offers almost no protection against an automated attack.

91%

Of people reuse passwords

One leaked password means every account that shares it is compromised. Unique, strong passwords for each service are the single most effective security habit you can build.
